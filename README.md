# タイタニック号 生存予測モデル

Kaggle Titanicコンペティション用の生存予測機械学習モデル

## 📊 概要
タイタニック号の乗客データを使用して、生存率を予測する機械学習モデルです。
ランダムフォレストアルゴリズムと特徴量エンジニアリングにより、高精度な予測を実現しています。

## 🎯 モデル性能
- **アルゴリズム**: ランダムフォレスト分類器
- **クロスバリデーション精度**: 82.83% (±0.86%)
- **使用特徴量**: 
  - 基本情報: 客室クラス、性別、年齢、運賃、乗船港
  - エンジニアリング特徴量: 家族人数、単独乗船フラグ、敬称、年齢区分、運賃区分

## 📁 ファイル構成
- `code.py` - メインの予測スクリプト
- `train.csv` - 訓練用データセット
- `test.csv` - テスト用データセット
- `submission.csv` - Kaggle提出用の予測結果
- `README.md` - プロジェクト説明（本ファイル）
- `.gitignore` - Git管理除外設定

## 🚀 使い方

### 必要なパッケージのインストール
```bash
pip install pandas numpy scikit-learn
```

### 予測の実行
```bash
python code.py
```

実行すると以下が出力されます：
- モデルの精度評価（クロスバリデーション結果）
- 予測結果のサマリー
- `submission.csv` の生成

## 📈 予測結果
- 生存予測数: 156人 / 418人
- 予測生存率: 37.32%

## 🔧 特徴量エンジニアリング
1. **家族サイズ** - 兄弟姉妹・配偶者数 + 親・子供数 + 本人
2. **単独乗船フラグ** - 家族がいない乗客の識別
3. **敬称抽出** - 名前からMr/Mrs/Miss/Master等を抽出・分類
4. **年齢区分化** - 年齢を5つのグループに分割
5. **運賃区分化** - 運賃を4つのグループに分割
6. **カテゴリ変数の数値化** - 性別・乗船港を数値に変換
